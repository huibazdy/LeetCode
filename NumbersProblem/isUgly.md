## 丑数（263）

### 问题描述

丑数是只包含**质因数**为`2、3、5`的正整数。一个例外是：习惯上将`1`视为第一个丑数。前十个丑数分别是：

`{1,2,3,4,5,6,8,9,10,12}`



### 问题分析

根据丑数的定义，其一般性表达式（n>1时）为：
$$
2^a + 3^b + 5^c(a,b,c\in N)
$$
可以不断地用2,3,5来除n，如果最后的结果是1，则说明是丑数，若最后不是1则说明不是。



### 问题解法

```c++
bool isUgly(int n)
{
    if(n <= 0)
        return false;
    if(n == 1)
        return true;
    while(n > 1)
    {
        if(n % 2 == 0)
            n = n / 2;
        else if(n % 3 == 0)
            n = n / 3;
        else if(n % 5 == 0)
            n = n / 5;
        else
            return false;
    }
    return true;
}
```
