# Dynamic Porgramming

> 穷举：如何聪明地穷举？

* 子问题
* 重叠子问题
* 最优子结构
* DP 状态：每个子问题的状态
* 状态转移方程
* 自底向上：每个子问题之间有联系，这种联系就是递推公式，也称为状态转移方程；自下而上可以消除大量重叠子问题。（递归是自顶向下）
